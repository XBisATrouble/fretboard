# Guitar samples

The bundled `acoustic_guitar_nylon-mp3.js` contains a reduced E2–E6 sample set
from FluidR3_GM, sourced from
[midi-js-soundfonts](https://github.com/gleitz/midi-js-soundfonts).

FluidR3_GM is licensed under the
[Creative Commons Attribution 3.0 license](https://creativecommons.org/licenses/by/3.0/us/).
Only seventeen anchor samples are bundled; intermediate pitches are produced
with Web Audio playback-rate transposition.
